Функции админа:
1)	Загрузить в приложение интерфейса json-файл. Такой же файл должен грузиться в Графану.
2)	Раздать права пользователям, лучше через AD. Права на посчитать и на сохранить должны различаться.

Функции пользователя:
1)	Загрузить на экран схему своей станции вместе с текущими измерениями
2)	Получить оценку режима. 
3)	Изменить измерения вручную
4)	Загрузить последние или дефолтные оперативные ограничения для котлов и турбин
5)	Изменить оперативные ограничения для котлов и турбин.
6)	Включить/отключить котлы, турбины и РОУ.
7)	Включить/отключить ПВД для включенных турбин
8)	Полученный набор измерений, ограничений и состояний отправить в оптимизатор и получить для данного набора расходную характеристику, а также режим для 11 точек электрической нагрузки, включая Рмин и Рмакс.
9)	Если результат устраивает, сохранить его в БД с соответствующим числом и именем состава.
10)	Если не устраивает, поменять еще что-то, отправить, посмотреть, до тех пор, пока результат не устроит.
