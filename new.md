Для раскраски ПСГ по п.4 прошлого собрания добавил по ТА 2-4 переменные:
•	T2.PSG1_On == 1: ПСГ-1 у ТА-2 включен
•	T4.PSG2_On == 0: ПСГ-2 у ТА-4 выключен
•	И т.п. для 2-4 турбин
•	Для ТА-1 одна переменная T1.PSG_On. Тогда при 1 оба бойлера ТА-1 включены, при 0 – оба выключены. Работают параллельно, различить режимы нет возможности.
Датафреймы meas, seac, opt1, opt2 добавил в архив во вложении.
На SVG-картинке ПСГ и так разделены, осталось только их покрасить по новым переменным.
